
# Xi Agency

> Un site web de marketing digital moderne, conçu pour présenter les services d'une agence spécialisée en web design, branding et marketing digital.

## Fonctionnalités principales
- Présentation de l'agence et de ses services
- Slider interactif (Owl Carousel)
- Design responsive (Bootstrap)
- Section portfolio/projets
- Contact et réseaux sociaux

## Installation
1. Clonez ce dépôt ou téléchargez-le en ZIP.
2. Ouvrez le dossier dans votre éditeur ou navigateur.
3. Lancez `index.html` pour voir le site en local.

## Technologies utilisées
- HTML5, CSS3, JavaScript
- [Bootstrap](https://getbootstrap.com/)
- [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/)

## Structure du projet
- `index.html` : page principale
- `css/` : styles (Bootstrap, Owl Carousel, style personnalisé)
- `js/` : scripts (jQuery, Bootstrap, Owl Carousel, scripts personnalisés)
- `img/` : images et logos
